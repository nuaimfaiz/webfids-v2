# WebFIDS - Flight Information Display System

A modern, fully functional flight information board built with Next.js, TailwindCSS, and simulated real-time data. Designed for instant deployment on Vercel (100% free hosting).

## Features

- 🛫 **Departures & Arrivals** – Toggle between boards with a clean tab interface
- 🔍 **Live Search** – Filter flights by flight number, destination/origin, or airline
- ⏱ **Dynamic Status** – Flights automatically update status (On Time, Delayed, Boarding, Departed/Landed) based on current time and simulated delays
- 📱 **Responsive Design** – Table view on desktop, card view on mobile
- 🎨 **Modern UI** – Dark-themed FIDS aesthetic with subtle animations and real-time clock
- 🚀 **100% Free Hosting** – Optimized for Vercel, no external API keys required

## Deployment Instructions

1. Push this code to a GitHub repository
2. Log in to [Vercel](https://vercel.com)
3. Click "Add New Project" and import your repository
4. Keep all default settings – Vercel will automatically detect Next.js
5. Click "Deploy" – your WebFIDS will be live within minutes!

No environment variables or build commands are needed. The mock flight data and status logic are entirely self-contained.
